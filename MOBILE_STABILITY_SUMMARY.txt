KNOWN LIMITATIONS (BY DESIGN)
================================================================================

Intentionally Disabled:
  ❌ Vertical scrolling (by design)
  ❌ Horizontal scrolling (by design)
  ❌ Pinch zoom (by design)
  ❌ Double-tap zoom (by design)
  ❌ Pull-to-refresh (by design)
  ❌ Overscroll bounce (by design)
  ❌ Long-press context menu (by design)
  ❌ Text selection (by design)

Why:
  The app is designed as a full-viewport chat interface that should be
  static with all content visible without scrolling. This is the intended
  user experience.

If You Need to Change:
  - To enable scrolling: Remove position:fixed from html/body
  - To enable zoom: Remove user-scalable:no from viewport
  - To enable text selection: Remove user-select:none from CSS
  - See MOBILE_TESTING_GUIDE.md for modification instructions
